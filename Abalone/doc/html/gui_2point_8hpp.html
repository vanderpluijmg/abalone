unsigned  n,
T  radius,
T  arg_shift = 0 
)

Creates an orbit of n points symetrically spread around some coordinate, starting from an optional angle shift.

For instance, orbit({1,1}, 6, 2., (3.141592654 / 4)); creates 6 points around the coordinate (1,1), spreaded symetrically as a regular hexagon, the first point starting at an amplitude of pi/4.

The created points are all expressed in cartesian coordinates

Parameters
pthe point to orbit around
nthe number of points in the orbit
radiusthe radius of the orbit
arg_shitthe starting angle of the first orbiting point, specified in radians
Returns
a std::vector of n orbiting points
